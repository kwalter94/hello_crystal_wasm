# hello_crystal_wasm

Checking out wasm in Crystal!!!

## Installation

```sh
$ git clone git@github.com:kwalter94/hello_crystal_wasm.git
$ sudo zypper install lld15 # Installs wasm-ld on Suse Tumbleweed
$ shards install
$ make
```

## Usage

```sh
$ ./server # Go to http://localhost:3000 in your browser!!!
```
